## Framework de Tono Roure

### La voz de Roure es:

**Suave y poética** — "El cardigan que ibas a comprar tarde o temprano"
**Cercana como una amiga** — "Chicas, no puedo con la perfección de este look"
**Aspiracional pero accesible** — "Para las noches que merecen un outfit a la altura"
**Con referencias románticas y francesas** — nombres como Douceur, Claire, Celine, Odette

### Frameworks de Copy (adaptados de kostja94/marketing-skills al tono Roure)

El framework se elige según el objetivo de la pieza:

- **PAS** (Problema → Agitación → Solución) — cuando la clienta tiene una objeción ("¿No sabés qué ponerte?")
- **AIDA** (Atención → Interés → Deseo → Acción) — captions de lanzamiento de producto o drop
- **BAB** (Antes → Después → Puente) — UGC, testimoniales, reshare de clientas
- **FAB** (Features → Ventajas → Beneficios) — descripciones de producto en la tienda
- **Emocional Roure** (Contexto → Deseo → Micro-validación → CTA suave) — default para TikTok e Instagram

### Fórmula Emocional Roure (default)

```
[Contexto emocional — dónde se ve usando esto]
[La prenda, descrita desde el deseo, no desde la ficha técnica]
[Micro-validación — "es exactamente eso que necesitabas"]
[CTA suave, nunca agresivo]
```

### Ejemplos de Frameworks Aplicados a Roure

**PAS — Para caption con objeción de "no tengo qué ponerme":**

```
¿Ese momento en que abrís el placard y no sabés qué ponerte? 🧸
[PROBLEMA]
Sí, ese. El de todos los lunes.
[AGITACIÓN]
El Cardigan Dolce lo resuelve. Siempre queda bien, con todo.
[SOLUCIÓN]
Link en bio 🌷
```

**BAB — Para reshare de clienta:**

```
Antes: "no sabía cómo armar looks de invierno"
Después: este outfit que no se quiere sacar 🩷
El puente: Sweater Bear + Wide Leg Sastrero de @missroure
¿Cuál es tu combo favorito?
```

**AIDA — Para lanzamiento de drop:**

```
Llegó lo que estabas esperando ✨  [ATENCIÓN]
La nueva colección de invierno de Roure — suave, romántica, tuya.  [INTERÉS]
Cada prenda elegida para que te veas linda sin esfuerzo.  [DESEO]
Link en bio para ver todo 🌷  [ACCIÓN]
```

## Reglas de Escritura

### ✅ HACER
- Voseo: "te vas a querer sacar", "lo que ibas a comprar"
- Frases incompletas con intención: "Soft, romántica, tuya."
- Preguntas que invitan a soñar: "¿Qué harías vos con este look?"
- Emojis estratégicos al final de frase: 🌷 🧸 🩷 ✨
- Mencionar el descuento de transferencia cuando es contenido de producto
- En TikTok: precio + WhatsApp en caption

### ❌ NO HACER
- "¡ÚLTIMA OPORTUNIDAD! ¡OFERTAZO! ¡NO TE LO PIERDAS!"
- Urgencia forzada o escasez falsa
- Lenguaje corporativo o impersonal
- Anglicismos innecesarios que rompan el registro cálido
- Exceso de signos de exclamación
- Describir la prenda técnicamente sin emoción ("tela 95% algodón…")

## Templates de Copy por Canal

### TikTok — Caption (máx 150 caracteres para el hook)

```
[Nombre del producto] [emoji Roure]
[Precio / oferta explícita]
[WhatsApp]
[Hashtags: 4-5]
```

### Instagram — Caption de Carrusel

```
[Frase hook emocional — 1 línea]

[Descripción suave del producto — 2-3 líneas]
[Pregunta abierta para comentarios]

[CTA: link en bio / WhatsApp]

[Hashtags: 8-12 específicos del nicho]
```

### Descripción de Producto para Tienda

```
[Nombre del producto]

[1 frase emocional de apertura]
[Descripción de cómo se siente / cómo queda — desde la experiencia, no la ficha]
[Mencionar colores disponibles]
[Mencionar talle si es relevante]

Precio: $XX.XXX
Con transferencia: $XX.XXX (20% OFF)
```

### WhatsApp / Comunicación Directa

- Tono de amiga, no de bot
- Siempre empezar con el nombre si lo tenés
- Dar info clara de precio, talle, stock
- Cerrar con algo cálido: "¡Avisame si te surge alguna duda!" 🌷
