## Reglas de Escritura para Descripciones

1. **Siempre empezar con emoción, nunca con especificación técnica**
2. **Mencionar siempre el precio con transferencia** (20% OFF)
3. **Mencionar envío gratis si el producto + algún combo supera los $100.000**
4. **Incluir WhatsApp para consultas** — la tienda cierra, pero el WhatsApp fideliza
5. **Nunca usar frases genéricas** como "hermosa prenda de calidad"
6. **Voseo siempre** — "que no vas a querer sacarte", "el que ibas a comprar"
